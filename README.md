# Unnamed RNG Game

## UI Design
All UI panels are designed in Figma: https://www.figma.com/design/2thf67I1YzdA0XA1zyFDL0/

### Panels Designed:
- 🌿 Loading Screen (1920x1080) - earthy green, avatar, progress bar, PLAY button
- 🎲 Main HUD Roll Bar - ROLL/AUTO/QUICK buttons, coins, luck display
- 🎒 Inventory Panel - aura grid with rarity colors and stud texture
- 💬 NPC Dialogue Card - parchment style, straight viewport, choice buttons
- ⚙️ Settings Panel - tabbed settings with toggles
- ⛏️ Mining HUD - explosion ability + return to surface buttons

## Color Palette
- BG Dark: #201608 (earthy brown)
- Gold: #F5CC3C (accent)
- Green: #50C864 (primary action)
- Cream: #F0E6C8 (text)

## Game Theme
Bossa nova music, earthy mining/nature vibe, retro studded texture on all UI
